<h3>Requirements</h3>
For building and running the application you need:
 
 - Node.js 18.x
 - Npm 9.2.0
 
<h3>Installation</h3>

```
$ git clone https://github.com/WooK1184/Item_sim
$ cd Item_sim
$ cd server
```
#### Run Method
```
$ npm install
$ npm install dotenv
$ npm install prisma @prisma/client
$ npm install jsonwebtoken
$ node server.js 
```
